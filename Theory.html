One-Time Pad Cipher

Theory

Vernam proposed a bit-wise exclusive or of the message stream with a truely random zero-one stream which was shared by sender and receipient.(Vernam Cipher is also called one time pad(OTP).

In One Time Pad(OTP),a key should be used only once for particular cipher text.This makes execution more strong.The lenght of plaintext and key should be equal.

In above example we take plaintext as “add” and key as “daa” after XOR operation we Encrypted text (Cipher text) as “ddd”.


If above generated cipher as “ddd” XOR with key as “daa” we get our original plain text as “add”.
